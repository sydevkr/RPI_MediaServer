# 00README.md

- 처음 작업 시 아래 문서를 확인해 현재 개발 상태를 파악한다.
- `01REQUIREMENTS.md`: 요구사항
- `02ENVIRONMENT.md`: 개발환경, 개발 스펙, 설계 구조
- `03TASKS.md`: 현재 작업 상태
- `04ISSUES.md`: 현재 주의사항
- `05MANUAL.md`: 실행 방법, 확인 방법

## 문서 업데이트 원칙
- MD 파일 업데이트 시 최대한 간결하고 심플하게 표현한다.
- 같은 내용을 여러 문서에 반복해서 길게 쓰지 않는다.
- `03TASKS.md`는 상태만, `04ISSUES.md`는 이유/주의사항만, `05MANUAL.md`는 실행 방법만 기록한다.
