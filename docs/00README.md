- 처음 작업 시 아래 문서를 확인해 현재 개발 상태를 파악한다.
- `01REQUIREMENTS.md`: 요구사항

## 문서 업데이트 원칙
- MD 파일 업데이트는 원칙적으로 하지 않는다. 
- 개발자가 직접 수정하고 업데이트하고 관리한다. 
- 만약 개발자가 업데이트 요청이 있다면 최대한 심플하고 간결하게 표현한다.
